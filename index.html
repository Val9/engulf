engulf

Visualization tool for webserver concurrency, written in Clojure and Javascript.

Get up and running

  1. This only runs in google chrome at the moment
  2. Download the jar from the downloads page
  3. Run it like so PORT=4000 java -jar engulf-VERSION.jar
  4. Visit http://localhost:4000 in your browser to use the GUI

Hacking

Engulf is a work in progresss. The currently released version is the 2.x branch. Master represents the upcoming 3.0 version, which will feature distributed operation, saved results, a REST API, and numerous other improvements. It's currently only 70% done and barely runnable (with no UI). If you're interested in helping out, just ping me!

Legacy

There is a legacy version of engulf available on the downloads page and the legacy branch that runs quite differently.
